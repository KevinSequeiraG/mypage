# Kevin Sequeira — Developer Portfolio

Personal portfolio website showcasing my projects, work experience and skills as a Full Stack Developer.

🔗 **Live site:** https://kevinsequeirag.vercel.app

## Overview

A bilingual (English / Spanish) portfolio built as a single-page experience, featuring sections for projects, professional experience, background and a contact area. Designed to give recruiters and collaborators a quick, clear view of my work.

## Tech Stack

- **Framework:** Next.js
- **Language:** JavaScript
- **Styling:** Tailwind CSS
- **i18n:** next-i18next (English / Spanish)
- **Animations:** AOS / Animate.css
- **Deployment:** Vercel

## Features

- 🌐 Full bilingual support (EN / ES)
- 📂 Projects, experience and skills sections
- 📱 Responsive, animated UI
- 📨 Direct contact links (email, LinkedIn, GitHub)

## Running locally

```bash
git clone https://github.com/KevinSequeiraG/mypage.git
cd mypage
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
